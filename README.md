[==Intoduction==]

  This is an Android App Project using Kotlin.
  
  You could use it as Reminder of Week Routines. The Data of Routines are set in Project code. 
  
  You could just modify the routine data and regenerate the app(.apk file). Or add one API interface to get data from online database by yourself.

[==Features==]

(ver.1)

  LongClick routine could complete ths routine. Click routine could see the details.

(ver.2)

  Show button which could update listview data to today when resume app.

[==Why This App is Good==]

 (1) Can keep last complete date if always add new routine at end of data list after reinstall data.
 (2) Less in clean Code and more in our demand.
 (3) Friendy UI to the olders and people with big hands.
 (4) Great in privacy. Data only in local and phone. 
 (5) Also work if no Network.
